My Projects

Weather Wizard [Swift]

An iOS application which displays the current weather conditions of the entered region by making use of wunderground's. online API. The information (in JSON) returned on calling the API is parsed and the required fields (such as temperature in °C and °F, current conditions) are extracted as required.

TennisFun [JavaScript]

A simple tennis game built using JavaScript (embedded in HTML). The game was developed on an HTML canvas with a specific width and height. The motion of the ball in the game was mimicked by calling an update location function repeatedly (very short intervals of time) which pinpointed the location of the ball at that specific moment of time.

Digital Diary [C++]

A 1500 line C++ project which simulates the working of one of the older technologies known as a Digital diary . The model includes a password protected login page, a scientific calculator, a horroscope teller, a basic unit converter and a world clock. This was one of my high school projects in 2014.

Retro Calculator [Swift]

An iOS calculator application, the UI of which was built using StackViews and constraints. Custom sounds and fonts have been used to enhance the appeal of the application.
